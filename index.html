Welcome to the Twinlife GitHub Pages.

Twinlife services are available for the following native target environments:

  • Android: Java
  • iOS: Objective C
  • Web applications: Javascript and Web Components (Polymere)

For each target environment, Twinlife services are organized as a hierarchy along the following levels:

  • Twinlife Base Services: generic twinlife platform services
  • Twinlife Add-on Services: services specific to a particular application or extensions to the Twinlife Base Services.

Twinlife Base Services

Management Services

  • Configuration
  • Events
  • Notifications
  • Reports

Account Services

  • Sign Up
  • Sign In
  • Sign Out

Connectivity Services

  • Application connection/disconnection
  • To Server
  • To Internet

Roster Services (XMPP)

  • Contacts
  • Groups
  • Invitations

Chat Services (XMPP)

  • Text chat messages

Directory Services

  • Name to Object directory (LDAP style)

Repository Services

  • Object data base (based on PostgreSQL)
  • Distributed fine grain mobile objects management

WebRTC Services

  • Peer connection
  • Media stream
  • Data channel

Twincode Services

  • Twincode Factory
  • Twincode IN
  • Twincode OUT
  • Twincode Switch

Designer Templates

We've crafted some handsome templates for you to use. Go ahead and continue to layouts to browse through them. You can easily go back to edit your page before publishing. After publishing your page, you can revisit the page generator and switch to another theme. Your Page content will be preserved if it remained markdown format.

Rather Drive Stick?

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
